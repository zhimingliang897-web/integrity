## 1. 环境准备

### 你需要准备的东西

- 一台红米/小米安卓手机
- 一条能传输数据的USB数据线（充电线不一定行）
- 一台Windows电脑
- 已安装Conda（你已经有了）

---

## 2. 手机端配置

### 步骤2.1: 开启开发者模式

1. 打开手机「设置」
2. 进入「我的设备」
3. 点击「全部参数与信息」
4. **连续点击「MIUI版本」7次**
5. 看到提示「您已处于开发者模式」即可

### 步骤2.2: 开启USB调试

1. 返回「设置」主页
2. 进入「更多设置」
3. 进入「开发者选项」
4. 开启以下选项：
   - ✅ **USB调试**
   - ✅ **USB调试（安全设置）** ← 这个很重要！
   - ✅ **不锁定屏幕**（充电时保持亮屏）

### 步骤2.3: 连接电脑

1. 用USB数据线连接手机和电脑
2. 手机会弹出「是否允许USB调试」
3. 勾选「始终允许来自这台计算机」
4. 点击「确定」

### 步骤2.4: 大麦APP准备

1. 打开大麦APP
2. 登录你的账号
3. 进入「我的」→「观演人」
4. 点击「添加观演人」
5. 填写真实姓名和身份证号（**必须与身份证完全一致**）
6. 保存观演人信息

## 3. 电脑端配置

### 步骤3.1: 安装ADB工具

1. 访问: https://developer.android.com/tools/releases/platform-tools
2. 点击「Download SDK Platform-Tools for Windows」
3. 下载并解压到一个目录，例如: `C:\platform-tools`

### 步骤3.2: 添加环境变量

1. 按 `Win + R`，输入 `sysdm.cpl`，回车
2. 点击「高级」选项卡
3. 点击「环境变量」
4. 在「系统变量」中找到 `Path`，双击
5. 点击「新建」，输入: `C:\platform-tools`（你的解压路径）
6. 点击「确定」保存

### 步骤3.3: 验证ADB安装

打开命令提示符（按 `Win + R`，输入 `cmd`），运行:

```bash
adb version
```

如果显示版本号，说明安装成功。

### 步骤3.4: 创建Conda环境

```bash
# 进入项目目录
cd e:\integrity\16票

# 创建环境
conda env create -f environment.yml

# 激活环境
conda activate damai
```

### 步骤3.5: 初始化uiautomator2

首次使用需要在手机上安装一个辅助APP:

```bash
# 确保手机已连接
adb devices

# 初始化（会在手机上安装ATX应用）
python -m uiautomator2 init
```

手机上会弹出安装提示，请允许安装。

## 5. 配置抢票信息

编辑 `config.py` 文件:

```python
# === 抢票时间 ===
# 格式: "YYYY-MM-DD HH:MM:SS"
TARGET_TIME = "2026-03-20 10:00:00"

# === 购票数量 ===
TICKET_COUNT = 1

# === 实名观演人 ===
# 填写你在大麦APP中添加的观演人姓名
# 必须完全一致，包括空格
VIEWER_NAMES = ["张三"]

# 多人购票示例:
# VIEWER_NAMES = ["张三", "李四"]

# === 票价优先级 ===
# 脚本会按顺序尝试，优先买第一个
PRICE_PRIORITY = ["580", "380", "280"]

# === 场次优先级（可选）===
# 如果有多场，按优先级选择
# 如果只有一场或不在乎场次，留空: []
SESSION_PRIORITY = []
```

## 6. 运行抢票

### 抢票前准备（提前1天）

1. ✅ 确认大麦APP已登录
2. ✅ 确认观演人已添加
3. ✅ 记录演出的开票时间
4. ✅ 修改 `config.py` 配置

### 抢票当天

#### 第一步: 连接手机

1. 用USB线连接手机和电脑
2. 手机保持解锁状态
3. 打开大麦APP，进入目标演出详情页

#### 第二步: 运行脚本

```bash
conda activate damai
cd e:\integrity\16票
python main.py
```

#### 第三步: 等待

脚本会显示倒计时:
```
[*] 距离开票还有: 05:00
[*] 距离开票还有: 04:59
...
[!] 倒计时: 0.500 秒
```

#### 第四步: 自动抢票

开票瞬间，脚本会自动:
1. 点击「立即购买」
2. 选择票价
3. 选择观演人
4. 提交订单

#### 第五步: 手动支付

**抢票成功后，请立即在手机上完成支付！**

订单通常有15分钟的支付时限。

## 7. 常见问题

### Q: 设备连接失败

**检查项:**

1. USB线是否支持数据传输（有些只能充电）
2. USB调试是否开启
3. 是否授权了USB调试
4. ADB是否正确安装

**解决方法:**

```bash
# 检查ADB是否识别设备
adb devices

# 如果显示 unauthorized，在手机上重新授权
# 如果显示 offline，尝试:
adb kill-server
adb start-server
adb devices
```

### Q: 无法点击手机屏幕

**原因:** USB调试(安全设置)未开启

**解决:** 开发者选项 → 开启「USB调试（安全设置）」

### Q: 时间同步失败

**影响:** 不大，会使用本地时间

**建议:** 确保电脑时间是准确的

### Q: 找不到购买按钮

**原因:** 大麦APP界面更新

**解决:**
1. 确保停在演出详情页
2. 运行 `test_connection.py` 导出UI结构
3. 查看 `screenshots/test_ui_dump.xml` 找到正确的按钮文字

### Q: 抢票成功但未选中观演人

**原因:** 观演人姓名配置不正确

**检查:**
1. `config.py` 中的姓名是否与大麦APP中完全一致
2. 注意检查是否有空格

### Q: 热门演出还是抢不到

**客观事实:**
- 热门演出竞争非常激烈
- 自动化只能提高效率，不能保证成功
- 黄牛可能有更专业的设备和网络

**建议:**
- 使用稳定的网络
- 提前5分钟启动脚本
- 多设置几个票价选项

## 技术原理

本工具使用 uiautomator2 库通过ADB协议控制手机。工作流程:

1. 电脑通过USB连接手机
2. Python脚本发送UI操作指令
3. 手机上的ATX服务执行点击等操作
4. 模拟真实用户操作，不修改APP

这种方式比直接调用API更安全，不容易被平台检测。
