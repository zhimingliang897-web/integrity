# 🤖 个人文件助手 Agent

这是一个可以让你用“大白话”直接控制电脑搜索全盘文件、并能自动发送文件到邮箱的 **AI 私人助手**。
最棒的是，无论是你在回家路上、在公司摸鱼，都能通过手机直接访问自己远在天边的家里的电脑！

本项目将为你提供最傻瓜式的本地化大模型 Agent 体验，小白也能在 5 分钟内搭建完毕。

---

## 🌟 核心功能特性

1. **🚀 极速智能全盘检索：** 在几秒内帮你找出所有盘符、各个深层文件夹里被遗忘的“简历/报告/小视频”。（已自动跳过系统垃圾与代码缓存目录提升千倍速度）。
2. **👀 真正的全图视野：** 手机端会以极其美观的现代化界面，**实时显示**当前电脑哪块硬盘的哪个文件夹正在被扫描。
3. **📂 原生目录穿越：** 在聊天框直呼“看看D盘下有什么”，“发一下C盘文档目录”，即可随时点开探寻。
4. **📧 霸道总裁发邮箱：** 找到文件后，直接吩咐：“把前两个 mp4 发给老板 abc@qq.com”，Agent 自动打包发送！
5. **🪄 一键内网穿透：** 启动脚本连带自动映射专属独立域名，没有复杂路由设置，走到哪连到哪。
6. **⚙️ 可视化控制台：** 不用看枯燥的代码！新版自带纯 Web 界面进行各项密钥配置。

## 🛠️ 下白必看的 3 分钟部署指南

如果你根本不懂什么是 Python 和代码，照着下面一步步做，绝对能成功！

### 第一步：安装必备的“地基”（Python）
1. 前往 Python 官网下载界面 [Python.org](https://www.python.org/downloads/)。
2. 下载安装包并双击运行。
3. **【极其重要】**：在安装界面的第一页底部，千万别忘了勾选 **`Add Python to PATH`**（将 Python 添加到环境变量），然后一直点击下一步完成安装。

### 第二步：下载本项目
1. 在当前 GitHub 页面右上角绿色的 **`Code`** 按钮处点击，选择 **`Download ZIP`**。
2. 下载后，解压到你喜欢的任意文件夹里（比如 `D:\小文件助手`）。

### 第三步：复制一份配置文件
1. 打开解压好的文件夹，你会看到一个名为 `config.example.json` 的文件。
2. 请把它**复制并粘贴出另一份**，然后将新文件重命名为 **`config.json`**。（一定要有这个文件程序才能活过来！）

### 第四步：一键神启动！
1. 找到文件夹里那个名叫 **`start.bat`** （带有齿轮图标的批处理文件）。
2. **直接双击它！** 第一次运行时，它会自动去宇宙最深处下载所需的依赖库（大概要等一两分钟）。
3. 等黑窗口里出现 `文件助手 Agent 已启动！` 字样时，说明本地已经成了！别关掉任何黑窗口！

## 🔑 第五步：傻瓜式“配置界面”教学（只需填一次）

当上一步看到成功后，请在你的电脑浏览器里打开这个临时网址：
👉 [http://localhost:5000/settings](http://localhost:5000/settings)

你在这个精美的网页里填入参数之后，你的大模型和内网穿透就会完全激活。

#### 1. 阿里云百炼 (DashScope API Key)
这个助手的心脏是大模型！咱们用的“通义千问”免费且好用：
- 登录 [阿里云百炼控制台](https://bailian.console.aliyun.com/)。
- 开通后，点击左侧栏的“API-KEY 管理”，创建一个新 Key。
- 复制这个长得像 `sk-xxxxxx` 的秘密放入网页的第一个框。

#### 2. 内网穿透 (Natapp Authtoken)
为了让你出门在外手机也能访问，不用记难搞的 IP，我们用 Natapp：
- 登录 [Natapp 官网](https://natapp.cn/) 并注册。
- 点击左侧“购买隧道” -> “免费购买” -> 随便起个名字拿免费配额（如果想要固定网址可以花钱买 VIP_1）。
- 把那个类似 `2f2ad52xxxxxxxxx` 的 Authtoken 复制到网页设置的第二个框！

#### 3. 霸道发件邮箱（QQ 邮箱演示）
- 电脑登录 [QQ 邮箱网页版](https://mail.qq.com/)。
- 顶部点击 `设置` -> `账户`。
- 向下找到 `POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务`，点击开启 **SMTP 服务**。
- 按要求发个短信，它会给你一串 16 位的**纯英文字母密码**（例如：`geqzijwnfsqidebd`）。这就是“邮箱授权码”。
- 把你的 QQ 邮箱账号 和 这16位授权码 填进最后两个框。

### 🎉 终章：保存并重启
一切在网页上点击保存完成后：
1. **关掉你刚才打开的所有的黑窗口！！！**
2. **重新双击运行一次 `start.bat`。**

这时弹出的黑窗口里会赫然出现这样一行魔法字：
> `Forwarding http://xxxxx.natapp1.cc -> 127.0.0.1:5000`

**用你手机里的任意浏览器，输入上面这串专属网址，开始体验你的私人 AI 文件大管家吧！** 🥂

## 防失联技巧 (高级省电绝招)
出门在外人不在家怎么连？
不要让你的电脑进入**“睡眠状态”**（这会让整个宇宙失去信号断线），最好的做法是：
在 Windows 电源设置中 把显示器设为 3 分钟后黑屏，把睡眠时间设置为 **“从不(Never)”**。现代电脑待机功耗比灯泡还低，放心敞开用！
