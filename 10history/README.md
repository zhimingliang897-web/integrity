# organize_media.py - 照片/视频去重整理工具

从多个文件夹中扫描照片和视频，**求并集、去重、按拍摄日期整理**。只复制，不移动/删除源文件。

## 它做了什么

```
输入:                           输出:
F:\相册备份1\  ──┐              output/
F:\相册备份2\  ──┼─ 去重+整理 → ├── 2018/
F:\手机导出\   ──┘              │   ├── 2018-02/
                                │   │   ├── 2018-02-11 184617.jpg
                                │   │   └── 2018-02-23 143957.jpg
                                │   └── 2018-08/
                                │       └── ...
                                ├── 2023/
                                │   └── ...
                                ├── _未知日期/        ← 无法确定拍摄时间的文件
                                │   └── screenshot_abc123.png
                                └── _重复文件/        ← 被去重的文件（仅供参考）
                                    ├── IMG_0001.jpg
                                    └── _重复对照清单.txt
```

- 源目录不改动，目标目录生成按 年/年-月 的归档层级
- `_重复文件/` 是重复文件的副本，并附对照清单，方便复核

## 快速开始

```bash
# 1) 安装依赖
pip install exifread pillow pillow-heif tqdm

# 可选：安装 FFmpeg (提供 ffprobe) 以读取视频拍摄时间
# Windows:  choco install ffmpeg   或下载官方包后把 ffprobe 加到 PATH
# macOS:    brew install ffmpeg

# 2) 先 dry-run 预览（不复制）
python organize_media.py F:\相册1 F:\相册2 -o E:\整理后 --dry-run --verbose

# 3) 确认无误后执行
python organize_media.py F:\相册1 F:\相册2 -o E:\整理后
```

### 参数

- SOURCE（位置参数）：一个或多个源文件夹路径
- -o / --output：输出目录路径（必需）
- --dry-run：仅打印将要执行的操作，不实际复制
- --verbose：打印每个文件的处理详情

## 支持的文件类型

- 照片：jpg、jpeg、png、heic、heif、tiff、tif、bmp、webp、gif
- 视频：mp4、mov、avi、mkv、wmv、flv、m4v、3gp、mts、m2ts

## 日期识别策略

按优先级依次尝试：

| 优先级 | 方法 | 示例 |
|--------|------|------|
| 1 | EXIF 元数据 | 照片的 DateTimeOriginal |
| 1 | ffprobe | 视频的 creation_time |
| 2 | 文件名解析 | `2018-02-11 184617.jpg` |
| 2 | 文件名时间戳 | `1464689880000.jpeg` |
| 2 | 复合时间戳 | `1681371254840_1737332047006_77.jpeg` |
| - | 无法确定 | 放入 `_未知日期/` |

> 不使用 mtime 伪装拍摄时间；无法确定时直接入 `_未知日期/`。

## 去重算法

三级过滤，效率优先：

1. 文件大小分组（跳过 60-70%）
2. 前 4KB 快速哈希
3. 8MB 分块的完整 MD5

同一内容的多份文件中，**保留文件名更能反映日期信息的那份**（如含“2018-xx-xx”或合理时间戳）。

## 验证与统计

- 验证重复文件是否有对应原件：
  ```bash
  python verify_dup.py E:\整理后
  ```
- 统计目录下文件数量（按扩展名）：
  ```bash
  python count_files.py E:\整理后
  ```

## 常见问题

- HEIC/HEIF 无法读取日期  
  安装 `pillow-heif`；若仍无法解析，则回退到文件名解析或收进 `_未知日期/`。
- 视频没有显示拍摄时间  
  安装 FFmpeg 并确保 `ffprobe` 在 PATH 中。
- 路径过长导致复制失败（Windows）  
  尽量缩短输出目录的路径深度，或在系统开启长路径支持。
