建立信任——不说‘trust us’
❌ 中式英语
请您相信我们…
Please trust us...
✅ 地道表达
We only work with teams who’ve already tried [Tool X] — so no pitch, just a real-time workflow demo if you’re curious.
我们只对接已用过[某工具X]的团队——所以不推销，感兴趣的话可直接看实时操作演示。
💬 实战对话
We only work with teams who’ve already tried Notion — so no pitch, just a real-time workflow demo if you’re curious.
我们只对接已用过Notion的团队——所以不推销，感兴趣的话可直接看实时操作演示。
We use Notion daily — send over the demo link!
我们天天用Notion——快把演示链接发来！
💡
用筛选条件（only work with…）暗示专业门槛，比‘trust us’有力10倍
