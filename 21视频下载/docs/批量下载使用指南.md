# 批量下载器 - 快速使用指南

## 🚀 一分钟上手

### 场景：NTU Learn 课程有 10 个视频，想一次性全下载

**步骤：**

1. **双击运行** `批量下载视频.bat`（Windows）或 `python batch_downloader.py`（macOS）

2. **粘贴课程页面链接**
   ```
   请粘贴课程页面链接（包含多个视频的页面）:
   > https://ntulearn.ntu.edu.sg/ultra/courses/_2701248_1/outline
   ```

3. **设置并发数**（建议3个）
   ```
   同时下载几个视频？(1-5) [默认: 3]: 3
   ```

4. **浏览器会自动打开**
   - 如果需要登录NTU，请完成登录
   - 登录完成后，回到终端按回车

5. **自动扫描和收集**
   - 脚本会自动找到所有视频
   - 依次点击播放，收集真实地址
   - 你可以在浏览器里看到自动操作过程

6. **选择下载方式**
   ```
   收集到的视频列表：
   ✅ [1] Week 1 - Introduction
   ✅ [2] Week 2 - Data Structures
   ✅ [3] Week 3 - Algorithms
   ...
   
   选择下载方式：
     1. 全部下载
     2. 选择部分下载
     3. 退出
   
   请选择 (1-3) [默认: 1]: 1
   ```

7. **坐等下载完成**
   - 3个视频会同时下载
   - 进度会实时显示
   - 所有视频保存在 `downloads/` 目录

## 💡 高级技巧

### 选择性下载

如果只想下载部分视频：

```
请选择 (1-3) [默认: 1]: 2

请输入要下载的视频编号（用逗号分隔，如: 1,3,5-8）: 1,3,5-8
```

这样只会下载第 1, 3, 5, 6, 7, 8 个视频。

### 断点续传

如果下载中途中断（网络断开、电脑关机等）：

1. 重新运行 `批量下载视频.bat`
2. 粘贴**同样的课程链接**
3. 会提示：
   ```
   发现之前的下载进度！
   是否继续之前的进度？(y/n) [默认: y]: y
   ```
4. 按回车，会跳过已下载的视频，继续未完成的

### 调整并发数

- **网速快、服务器稳定**：设置 5 个
- **平衡速度和稳定性**：设置 3 个（推荐）
- **担心被限速**：设置 1-2 个

## 🔧 故障排除

### 问题1：扫描不到视频

**原因**：页面结构特殊，自动扫描失败

**解决**：
- 脚本会自动进入手动模式
- 在浏览器里手动点击视频播放
- 脚本会监听并捕获视频流

### 问题2：某些视频收集失败

**原因**：视频加载慢或需要特殊操作

**解决**：
- 查看 `batch_progress.json` 文件
- 找到失败的视频编号
- 使用 `嗅探视频.bat` 单独下载那个视频

### 问题3：下载速度慢

**原因**：并发数太低或网络限速

**解决**：
- 增加并发数到 5
- 检查网络连接
- 尝试使用VPN

### 问题4：需要登录权限

**原因**：视频需要登录才能访问

**解决**：
1. 先运行 `python get_cookies.py`
2. 导出浏览器的登录凭证
3. 再运行批量下载器

## 📊 性能对比

| 方式 | 10个视频耗时 | 优点 | 缺点 |
|------|-------------|------|------|
| 手动一个一个下载 | ~50分钟 | 可控 | 太慢，需要人工操作 |
| 嗅探器逐个下载 | ~30分钟 | 稳定 | 需要手动点击每个视频 |
| **批量下载器（并发3）** | **~15分钟** | **全自动，快3倍** | 需要稳定网络 |
| **批量下载器（并发5）** | **~10分钟** | **全自动，快5倍** | 可能被限速 |

---

## 🎯 最佳实践

1. **首次使用**：先用 `嗅探视频.bat` 测试单个视频，确保能正常捕获
2. **获取Cookie**：如果课程需要登录，先运行 `get_cookies.py`
3. **选择合适的并发数**：建议从 3 开始，根据实际情况调整
4. **保持网络稳定**：批量下载期间不要断网
5. **定期清理**：下载完成后，可以删除 `batch_progress.json` 释放空间

---

## 📝 文件说明

- `batch_downloader.py` - 批量下载器主程序
- `batch_progress.json` - 进度保存文件（自动生成）
- `downloads/` - 视频保存目录
- `cookies.txt` - 登录凭证文件（可选）
