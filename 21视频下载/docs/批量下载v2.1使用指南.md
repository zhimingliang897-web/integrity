# 批量下载器 v2.1 使用指南

## 🎯 核心改进

v2.1 版本彻底解决了 **token 过期问题**，这是 NTU Learn 批量下载的最大障碍。

### 问题背景

NTU Learn (Kaltura/Panopto) 的视频 URL 是这样的：
```
https://...kaltura.../playManifest/...?ks=djJ8MjU...&expire=1234567890
```

这个 `ks` (Kaltura Session) 参数会在 **5-10 分钟**内过期。

**旧版本的问题**：
1. 收集视频 1 的 URL（有效期 5 分钟）
2. 收集视频 2 的 URL（有效期 5 分钟）
3. ...收集 10 个视频（耗时 10 分钟）
4. 开始下载视频 1 → ❌ **token 已过期！**

**v2.1 的解决方案**：
1. 收集视频 1 的 URL
2. **立即下载视频 1** ✅
3. 收集视频 2 的 URL
4. **立即下载视频 2** ✅
5. ...

## 🚀 快速开始

### 1. 运行脚本

```bash
python batch_downloader.py
```

### 2. 粘贴课程链接

```
请粘贴课程页面链接（包含多个视频的页面）:
> https://ntulearn.ntu.edu.sg/ultra/courses/_2701248_1/outline
```

### 3. 选择模式

```
选择下载模式：
  1. 边收集边下载（推荐，避免 token 过期）⭐
  2. 先收集后下载（传统模式，可能遇到 token 过期）

请选择 (1-2) [默认: 1]: 1
```

**推荐选择模式 1**（边收集边下载）

### 4. 登录并等待

- 浏览器会自动打开
- 如果需要登录，完成 NTU 账号登录（包括 2FA）
- 登录完成后，按回车继续

### 5. 自动处理

脚本会自动：
1. 扫描页面上的所有视频
2. 依次点击每个视频
3. 收集视频流 URL
4. **立即下载**（趁 token 还有效）
5. 保存进度

### 6. 完成

```
======================================================================
  全部完成！
  成功: 10 个
  失败: 0 个
  保存位置: downloads/
======================================================================
```

## 📋 两种模式对比

|------|---------------------|-------------|

| 适用场景 | NTU Learn、需要登录的网站 | YouTube、B站等 |
| 选择性下载 | ❌ 不支持 | ✅ 支持 |

## 🐛 故障排除

### 问题 1：找不到视频

**症状**：
```
⚠️  未找到视频元素
```

**原因**：
- 页面结构特殊
- 视频是动态加载的

**解决**：
- 等待页面完全加载后再按回车
- 手动滚动页面，让所有视频加载出来

### 问题 2：收集失败

**症状**：
```
❌ 未捕获到视频流
```

**原因**：
- 没有点击播放按钮
- 视频加载太慢

**解决**：
- 在浏览器里手动点击视频播放按钮
- 等待更长时间（脚本会等待 30 秒）

### 问题 3：下载失败

**症状**：
```
❌ 失败: 下载失败 (重试3次)
```

**原因**：
- 网络问题
- Token 过期（传统模式）
- 视频被删除

**解决**：
- 检查网络连接
- 使用边收集边下载模式
- 手动检查视频是否存在

### 问题 4：跨域 iframe 错误

**症状**：
```
⚠️  无法访问 iframe 内容: ...
```

**原因**：
- Kaltura/Panopto 是跨域的

**解决**：
- 这是正常的，v2.1 已经改用外部点击
- 不影响功能，可以忽略

## 💡 最佳实践

1. **始终使用边收集边下载模式**（除非你确定 URL 不会过期）
2. **确保网络稳定**（下载大文件需要稳定的网络）
3. **不要关闭浏览器**（在收集阶段）
4. **定期检查进度**（查看 `batch_progress.json`）
5. **使用 Cookie**（如果需要高清画质）

## 🎓 NTU 学生专用技巧

### 下载整个课程的所有视频

1. 打开课程的 "Content" 或 "Outline" 页面
2. 确保所有视频都在同一个页面上（不要进入子页面）
3. 运行脚本，粘贴页面 URL
4. 选择模式 1（边收集边下载）
5. 等待自动完成

### 处理 2FA 认证

1. 脚本会打开浏览器
2. 输入账号密码
3. 完成手机 2FA 验证
4. 登录成功后，按回车继续
5. 脚本会自动处理剩余步骤

### 下载高清视频

```bash
# 1. 先获取 Cookie
python get_cookies.py

# 2. 再运行批量下载
python batch_downloader.py
# 脚本会自动使用 cookies.txt
```
