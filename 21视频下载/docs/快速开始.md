# 🚀 快速开始指南

5分钟快速上手视频下载器！

---

## 📋 前置要求

- Python 3.8 或更高版本
- 网络连接

---

## ⚡ 三步开始

### 步骤 1：安装依赖

```bash
# 安装核心依赖
pip install yt-dlp

# 如果需要嗅探功能（首次运行会自动安装）
pip install playwright
playwright install chromium

# 安装 ffmpeg（可选，用于合并视频流）
# macOS
brew install ffmpeg

# Windows: 从 https://ffmpeg.org/download.html 下载
```

### 步骤 2：选择工具

根据你的需求选择合适的工具：

| 需求 | 工具 | 命令 |
|------|------|------|
| 下载 B站/YouTube 视频 | 常规下载器 | `python main.py` |
| 下载 NTU Learn 单个视频 | 嗅探下载器 | `python sniffer.py` |
| 下载 NTU Learn 整个课程 | 批量下载器 | `python batch_downloader.py` |

### 步骤 3：开始下载

```bash
# 运行对应的工具
python main.py

# 粘贴视频链接
URL> https://www.bilibili.com/video/BV1xx411c7mD

# 等待下载完成
# 视频保存在 downloads/ 目录
```

## 💡 进阶技巧

### 技巧 1：获取登录权限

如果视频需要登录才能访问（如 B站大会员、NTU Learn）：

```bash
# 1. 先获取 Cookie
python get_cookies.py

# 2. 选择浏览器
方法1: 用 yt-dlp 内置功能导出（无需关闭浏览器）
  正在尝试从 chrome 导出...
  ✓ 成功！已保存 1234 字节到 cookies.txt

# 3. 再下载视频（会自动使用 cookies.txt）
python main.py
```

### 技巧 2：选择性下载

只下载部分视频：

```bash
python batch_downloader.py

# 选择：2. 选择部分下载
请选择 (1-3) [默认: 1]: 2

# 输入视频编号
请输入要下载的视频编号（用逗号分隔，如: 1,3,5-8）: 1,3,5-8

# 只下载第 1, 3, 5, 6, 7, 8 个视频
```

### 技巧 3：断点续传

下载中途中断后继续：

```bash
python batch_downloader.py

# 粘贴同样的课程链接
> https://ntulearn.ntu.edu.sg/ultra/courses/_2701248_1/outline

# 提示发现之前的进度
发现之前的下载进度！
是否继续之前的进度？(y/n) [默认: y]: y

# 自动跳过已下载的视频，继续未完成的
```

### 技巧 4：调整并发数

根据网速和服务器稳定性调整：

```bash
python batch_downloader.py

# 网速快、服务器稳定：设置 5
同时下载几个视频？(1-5) [默认: 3]: 5

# 平衡速度和稳定性：设置 3（推荐）
同时下载几个视频？(1-5) [默认: 3]: 3

# 担心被限速：设置 1-2
同时下载几个视频？(1-5) [默认: 3]: 2
```

## 🔧 常见问题

### Q: 提示找不到 yt-dlp？

```bash
# 安装 yt-dlp
pip install yt-dlp

# 验证安装
python -m yt_dlp --version
```

### Q: 浏览器无法打开？

```bash
# 安装 playwright 浏览器
playwright install chromium
```

### Q: 下载速度慢？

- 使用批量下载器（并发下载）
- 增加并发数到 5
- 检查网络连接

### Q: macOS 无法运行 .bat 文件？

```bash
# .bat 是 Windows 文件，macOS 直接运行 Python 脚本
python main.py
python sniffer.py
python batch_downloader.py
```

更多问题请查看 [故障排除文档](故障排除.md)
