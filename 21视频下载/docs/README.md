## 📁 项目结构

```
21视频下载/
│
├── 📄 核心脚本
│   ├── main.py                    # 常规下载器（B站、YouTube等）
│   ├── sniffer.py                 # 嗅探下载器（NTU Learn单个视频）
│   ├── batch_downloader.py        # 批量智能下载器 ⭐（整个课程）
│   ├── ntu_sniffer.py            # NTU专用嗅探器
│   └── get_cookies.py            # Cookie获取工具
│
├── 🪟 Windows 快捷启动
│   ├── 下载视频.bat
│   ├── 嗅探视频(解决NTU等大链接).bat
│   └── 批量下载视频.bat
│
├── 📚 文档目录 (docs/)
│   ├── 快速开始.md               # 5分钟快速上手指南
│   ├── 批量下载使用指南.md        # 批量下载详细教程
│   ├── 更新说明.md               # v2.0 版本更新内容
│   └── 故障排除.md               # 常见问题解决方案
│
├── 🔧 核心模块 (core/)
│   └── downloader.py             # 下载器核心逻辑
│
├── 🌐 Web界面 (web/)
│   ├── app.py                    # Flask Web服务
│   └── templates/                # HTML模板
│
├── 🔌 浏览器插件 (extension/)
│   ├── manifest.json             # 插件配置
│   ├── popup.html                # 插件界面
│   ├── background.js             # 后台脚本
│   └── content.js                # 内容脚本
│
├── 📦 配置文件
│   ├── requirements.txt          # Python依赖
│   ├── environment.yml           # Conda环境配置
│   └── .gitignore               # Git忽略文件
│
├── 💾 自动生成文件
│   ├── downloads/                # 视频保存目录
│   ├── cookies.txt              # Cookie文件
│   └── batch_progress.json      # 批量下载进度
│
└── 📖 README.md                  # 项目主文档
```

## 💡 使用技巧

### 提高效率

1. **使用批量下载器** - 速度提升 3-5 倍
2. **调整并发数** - 根据网速设置 3-5 个
3. **获取Cookie** - 下载需要登录的视频
4. **断点续传** - 中断后继续，不会重复

### 避免问题

1. **先测试单个视频** - 使用嗅探器测试
2. **保持网络稳定** - 批量下载期间避免断网
3. **定期清理进度文件** - 完成后删除 `batch_progress.json`
4. **查看文档** - 遇到问题先查看故障排除文档
