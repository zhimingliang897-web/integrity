📥 21视频下载器

一键安装说明 · 无需服务器

⚡ 快速安装（推荐）

1
双击运行 install.bat
位置：21视频下载/install.bat
脚本会自动检测 Python、安装 yt-dlp 并完成注册
2
打开浏览器，访问 chrome://extensions/
开启右上角的 开发者模式
3
点击 加载已解压的扩展程序
选择文件夹：21视频下载/extension
4
🎉 完成！打开任意视频页面，播放视频后：
• 右下角会出现 ⬇️ 下载视频 悬浮按钮
• 或点击浏览器工具栏的插件图标

🔧 手动安装（如 bat 脚本失败）

1. 安装依赖：pip install yt-dlp

2. 生成注册文件：参考 install.bat 内容手动执行

3. 注册表路径：HKCU\Software\Google\Chrome\NativeMessagingHosts\com.videodl.native

💡 使用方式

A
自动模式：页面播放视频后，右下角自动出现下载按钮，点击即下载
推荐
B
插件弹窗：点工具栏插件图标，选择检测到的视频流，点"开始下载"
备用
C
复制命令：如未安装 native host，点"复制命令"，粘贴到终端运行
无需安装任何东西，任何时候都能用
📂 所有视频默认下载到 21视频下载/downloads/ 文件夹
