# 快速开始 - 批量下载器 v2.1

## 🚀 5 分钟上手

### 1. 运行脚本

```bash
cd "21视频下载"
python batch_downloader.py
```

### 2. 粘贴链接

```
请粘贴课程页面链接（包含多个视频的页面）:
> https://ntulearn.ntu.edu.sg/ultra/courses/_2701248_1/outline
```

### 3. 选择模式（推荐模式 1）

```
选择下载模式：
  1. 边收集边下载（推荐，避免 token 过期）⭐
  2. 先收集后下载（传统模式，可能遇到 token 过期）

请选择 (1-2) [默认: 1]: 1
```

### 4. 登录并等待

- 浏览器自动打开
- 完成 NTU 登录（包括 2FA）
- 按回车继续

### 5. 自动完成

脚本会自动：
- 扫描所有视频
- 依次收集并下载
- 保存到 `downloads/` 目录

## 🎯 核心改进

v2.1 的**边收集边下载**模式：

```
旧版本（有问题）：
收集视频1 → 收集视频2 → ... → 下载视频1（token已过期❌）

新版本（正确）：
收集视频1 → 立即下载视频1✅ → 收集视频2 → 立即下载视频2✅
```
