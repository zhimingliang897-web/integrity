## 🛠️ 工具箱介绍

文件夹里有 3 个主力工具（我都做成了双击即开的 `.bat` 文件）：

### 1. 『常规下载器』 👉 `下载视频.bat`
这是**最常用、最快**的下载方式。
- **适用场景**：B站、YouTube、抖音、小红书、推特 等 **1000+** 常见网站。
- **怎么用**：
  1. 双击运行 `下载视频.bat`。
  2. 终端里提示 `URL>`，右键**粘贴视频链接**并回车。
  3. 全自动下载最高画质 mp4 格式，保存在 `downloads` 文件夹里。
  - *(支持批量！你可以在群里复制几十个链接，一起粘贴进去下！)*

---

### 2. 『嗅探下载器』💥 👉 `嗅探视频(解决NTU等大链接).bat`
这个是专门针对**顽固教学网站、套娃网页、权限视频**的终极杀器。
- **适用场景**：NTU Learn、Canvas网课、公司内部培训视频、或者任何 `下载视频.bat` 报错无法提取的网站。
- **工作原理**：它真的打开了一个浏览器去“看这个网页”。
- **怎么用**：
  1. 双击运行 `嗅探视频.bat`。它需要不到1分钟下载隐藏内核环境 `Playwright`（只在第一次运行）。
  2. 粘贴那个一长串的“大网页”链接。
  3. 它会**弹出一个干净的 Chrome 浏览器窗口**前往那个网页。
  4. ⚠️ **重点**：在网页里，如果弹出了 NTU 登录，你就手动输入账号密码、甚至手机 2FA 通过。然后找到视频，**别管三七二十一，点一下视频上的播放按钮！**（有的它会自动点，有的需要你点）
  5. 只要缓冲条动了，背后的黑框瞬间就会捕捉到类似 `.m3u8` 或 `/delivery/video` 的藏匿地址并提示：`[🎯 抓到了！]`
  6. 直接在黑框里敲个回车 `y`，直接开启最高画质合并下载！

---

### 3. 『获取 Cookie 工具』 👉 `get_cookies.py`
有些网站（如最高画质的B站、有些奇怪内部视频），下载器提示你没有权限。那是因为下载器不知道你的登录状态。
用这个**同步你的浏览器身份**给下载器。
- **使用方法**：直接双击或者 `python get_cookies.py`
- 它会自动从你的 Chrome 或 Edge 浏览器里导出你在各大网站（B站、YouTube等）的登录凭证成一个 `cookies.txt` 文件。
- 只要有这个文本，你再次使用 `下载视频.bat` 去下 B站，就能默认拿到你账号权限支持的 4K 甚至 8K 画质了！

## 🧑‍💻 FAQ（关于嗅探失败的问题）

**Q: 嗅探器弹窗等半天提示找不到视频退出？**
- **可能原因一**：页面让你输入密码（登录费时太长），它等不到你操作。最新的版本我已经把耐心提高到了 **3 分钟**！只要3分钟内你登进去并且按下播放键，就能截胡。
- **可能原因二**：登进去了但是没捕获？有些网页的视频不会“自动播放”，如果不动，它是不拉流的。所以你务必在网页里**按一下播放按钮**，听到出声音了，立马就被截了！一旦提示捕获成功，你可以随手关掉那个浏览器窗口。
