## 🎯 补全计划（按优先级排序）

### 第一阶段：快速上线（1-2周）

#### 1. Token 消耗对比工具 ⭐⭐⭐
- **源码位置**：`18tokens/`
- **核心功能**：对比中文/英文/图片在不同模型的 Token 消耗
- **技术栈**：Python + Gradio → 改为 Flask API
- **在线化难度**：🟢 易
- **预计工时**：2-3 天
- **实现步骤**：
  1. 提取核心计算逻辑
  2. 创建 `/api/tools/token-compare` 端点
  3. 前端在 tools.html 添加 UI
  4. 测试和部署

#### 2. 多模型对比平台 ⭐⭐⭐
- **源码位置**：`17ai-compare/`
- **核心功能**：同时向多个 LLM 发送问题，对比回复
- **技术栈**：Node.js + Express → 整合到 Flask
- **在线化难度**：🟢 易
- **预计工时**：3-4 天
- **实现步骤**：
  1. 将 Node.js 逻辑迁移到 Python
  2. 创建 `/api/tools/ai-compare` 端点（支持 SSE 流式输出）
  3. 前端添加多模型对比 UI
  4. 配置多个 LLM API Key

#### 3. 台词学习工具 ⭐⭐⭐
- **源码位置**：`2台词/`
- **核心功能**：扇贝单词导入 → 检索美剧台词 → TTS 语音生成
- **技术栈**：Flask（已有）
- **在线化难度**：🟢 易
- **预计工时**：3-4 天
- **实现步骤**：
  1. 整理现有 Flask 代码
  2. 创建 `/api/tools/dialogue-learning/*` 端点
  3. 前端添加单词上传和台词展示 UI
  4. 集成 edge-tts 语音生成

### 第二阶段：功能完善（2-4周）

#### 4. 图文互转工具 ⭐⭐
- **源码位置**：`9图生文文生图/`
- **核心功能**：
  - 图片 → 提示词（视觉理解）
  - 提示词 → 图片（AI 生图）
- **技术栈**：Python + DashScope/VolcEngine API
- **在线化难度**：🟡 中
- **预计工时**：4-5 天
- **实现步骤**：
  1. 创建 `/api/tools/image-prompt` 端点（图片转提示词）
  2. 创建 `/api/tools/prompt-image` 端点（提示词转图片）
  3. 前端添加图片上传和预览 UI
  4. 添加任务队列处理大图

#### 5. 小红书图片生成器 ⭐⭐
- **源码位置**：`20AIer-xhs/`
- **核心功能**：文章 → LLM 拆分 → 生成小红书风格多图
- **技术栈**：Python + Pillow + LLM
- **在线化难度**：🟡 中
- **预计工时**：4-5 天
- **实现步骤**：
  1. 创建 `/api/tools/xhs-generator` 端点
  2. 实现文章拆分和图片渲染
  3. 前端添加文章输入和图片预览
  4. 支持批量下载（ZIP）

#### 6. AI 辩论赛 ⭐⭐
- **源码位置**：`5AI辩论赛/`
- **核心功能**：多模型驱动辩论 + TTS 语音 + 视频导出
- **技术栈**：Flask + SSE + edge-tts
- **在线化难度**：🟡 中
- **预计工时**：5-7 天
- **实现步骤**：
  1. 创建 `/api/tools/ai-debate` 端点（SSE 流式辩论）
  2. 实现多模型切换和轮次控制
  3. 前端添加辩论展示 UI
  4. 可选：视频导出使用 Celery 异步队列

---

### 第三阶段：进阶功能（1-2月）

#### 7. 分镜视频生成器 ⭐ （可选）
- **源码位置**：`1分镜/`
- **核心功能**：AI 剧本 → 分镜图 → 配音 → 合成视频
- **技术栈**：Python + FFmpeg + edge-tts
- **在线化难度**：🔴 难
- **预计工时**：2-3 周
- **不建议在线化原因**：
  - 需要服务器安装 FFmpeg
  - 视频合成耗时长（>5分钟）
  - 输出文件大，带宽成本高
  - 高并发时服务器压力大
- **替代方案**：
  - 保持桌面版
  - 或使用云渲染服务（成本高）
  - 或仅在线化"剧本生成"部分
