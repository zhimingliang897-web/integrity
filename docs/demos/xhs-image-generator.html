快速开始

📦 安装依赖

pip install -r requirements.txt

⚙️ 配置 API Key

# config.yaml
api:
api_key: sk-xxxxxxxxxxxxxxxx
base_url: https://dashscope.aliyuncs.com/compatible-mode/v1
model: qwen3.5-plus

🚀 运行

# 完整生成（图片 + 评论区）
python scripts/main.py input/my_article.txt

# 只生成图片
python scripts/main.py input/my_article.txt --no-comments

输出结构

output/<文章名>/
├── slide_00_cover.png # 封面图
├── slide_01.png # 内容页 1
