📋 EasyApply 浏览器插件

一键填充简历表单，自动追踪投递记录，让求职更高效

核心功能

⚡ 一键填充

保存简历信息后，在任意招聘网站的简历填写页面点击"一键填充"，自动填入所有信息。支持基本信息、教育经历、实习/工作经历、项目经历、家庭成员等完整表单。

📝

多简历管理

支持创建多份简历配置（如校招版、社招版），一键切换使用。

📊

投递追踪

自动记录每次投递的公司、岗位、时间、链接，支持状态管理。

📥

Excel 导出

投递记录可导出为 Excel 表格，方便统计和管理求职进度。

💾

本地备份

每次保存自动下载 JSON 备份到本地，数据安全有保障。

安装指南

Chrome 浏览器安装步骤

1

打开扩展页面

在地址栏输入 chrome://extensions/

2

开启开发者模式

右上角开启「开发者模式」开关

3

加载扩展

点击「加载已解压的扩展程序」

4

选择文件夹

选择 11easyapply 文件夹

# 克隆项目
git clone https://github.com/zhimingliang897-web/integrity.git

# 扩展所在目录
integrity/11easyapply/

Edge 浏览器安装步骤

1

打开扩展页面

在地址栏输入 edge://extensions/

2

开启开发人员模式

左侧开启「开发人员模式」开关

3

加载扩展

点击「加载解压缩的扩展」

4

选择文件夹

选择 11easyapply 文件夹

💡 Edge 与 Chrome 使用相同的扩展 API，功能完全一致。

使用流程

📝

填写简历信息

点击插件图标，在「简历填充」Tab 填写个人信息、教育经历、工作经历等。

💾

保存简历

点击「保存当前简历」按钮，简历信息将存储在浏览器本地。

🌐

打开招聘网站

访问任意招聘网站的简历填写页面（如智联招聘、前程无忧、Boss直聘等）。

⚡

一键填充

点击插件中的「一键填充当前页面」按钮，自动填写所有表单字段。

📋

保存投递记录

填充成功后，弹窗确认保存投递记录，自动记录公司、岗位、时间。

📊

导出 Excel

在「投递记录」Tab 点击「导出 Excel」，下载完整的投递统计表。

支持的字段

👤

基本信息

姓名、性别、证件号、出生日期、手机、邮箱、居住地等

🎓

教育经历

学校、学历、专业、时间、排名、毕业论文等

💼

实习/工作经历

公司、职位、时间、联系人、工作内容描述

🔧

项目经历

项目名称、角色、时间、项目描述

🌐

语言能力

语言类型、证书/成绩（如英语四六级）

🏆

证书/获奖

证书名称、颁发机构、获得时间

👨‍👩‍👧

家庭成员

关系、姓名、工作单位、联系电话

📝

附加信息

自我评价、GitHub、博客等补充说明

技术实现

🔧

Manifest V3

使用最新的 Chrome 扩展标准，确保兼容性和安全性。

🎯

智能匹配

通过字段名称智能匹配表单元素，支持多种命名方式。

📦
