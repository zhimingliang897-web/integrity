工作流程

1

选择分类或输入主题

从预设分类中选择，或输入自定义主题如"咖啡店点单"

2

AI 生成内容

通义千问自动生成选题、知识点、英语表达和中文解释

3

自动生成图片

Pillow 渲染生成精美封面和内容页图片

4

预览与发布

预览生成效果，支持一键发布到小红书（公开/私密）

核心功能

🤖

AI 自动生成

智能生成选题、文案、知识点，无需人工编写内容。

🎨

精美封面

自动生成视觉吸引力强的封面和内容页图片。

📱

一键发布

Playwright 自动化控制浏览器，一键发布到小红书。

📝

内容管理

查看待发布和已发布内容，支持批量管理。

🔄

Cookie 自动刷新

自动打开浏览器登录，获取并更新 Cookie。

📚

Obsidian 集成

导出到 Obsidian 进行内容管理和二次编辑。

快速开始

📦 安装依赖

pip install -r requirements.txt
playwright install chromium

🚀 启动图形界面

python gui.py

浏览器自动打开 http://127.0.0.1:7860

💻 命令行使用

# 只生成内容
python main.py -c "餐饮美食"

# 自定义主题
python main.py --topic "咖啡店点单"

# 生成并私密发布
python main.py -c "交通出行" --publish --private
