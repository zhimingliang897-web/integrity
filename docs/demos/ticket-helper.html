自动点击购买按钮、选择场次、选择票价、确认订单。

使用流程

1

手机端准备

开启 USB 调试模式，登录大麦 APP，添加实名观演人信息

2

电脑端配置

安装 ADB 工具，创建 Conda 环境，安装依赖

3

连接设备

USB 连接手机，运行测试脚本验证连接

4

配置抢票信息

设置抢票时间、票价优先级、观演人姓名

5

运行脚本

提前 5 分钟运行，等待开票时间自动执行

6

完成支付

抢票成功后，在手机上完成支付流程

配置示例

# config.py

# 抢票时间（精确到秒）
TARGET_TIME = "2026-03-20 10:00:00"

# 观演人姓名（必须与大麦APP中一致）
VIEWER_NAMES = ["张三", "李四"]

# 票价优先级（从高到低）
PRICE_PRIORITY = ["580", "380", "280"]

# 抢票失败重试次数
MAX_RETRY = 3

注意事项

⏰ 提前测试

建议先用冷门演出测试完整流程，熟悉操作步骤。

📱 保持屏幕常亮

抢票过程中手机屏幕需保持常亮，避免锁屏影响操作。

🌐 网络稳定

确保网络连接稳定，建议使用 WiFi 环境。

⚠️ 风险提示

热门演出竞争激烈，自动化工具不保证 100% 成功。
