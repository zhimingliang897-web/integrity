核心功能

🚀

极速全盘检索

几秒内找出所有盘符、深层文件夹里的文件，自动跳过系统垃圾目录。

📱

手机远程访问

出门在外也能用手机访问家里电脑，实时查看扫描进度。

📂

目录穿越浏览

直接说"看看D盘有什么"，即可随时浏览任意目录。

📧

自动打包发邮件

找到文件后，直接吩咐"发给xxx@qq.com"，自动打包发送。

支持的指令示例

🔍
"帮我把所有名字带'简历'的 PDF 文件找出来"

快速部署

📦 步骤一：安装 Python

下载并安装 Python，记得勾选 Add Python to PATH

📥 步骤二：下载项目

git clone https://github.com/zhimingliang897-web/integrity.git
cd integrity/14file-agent-v0

⚙️ 步骤三：配置文件

复制 config.example.json 为 config.json

copy config.example.json config.json

🚀 步骤四：一键启动

双击运行 start.bat，首次运行会自动安装依赖

start.bat
💡 配置说明：启动后访问 http://localhost:5000/settings 进行可视化配置，填入：
  • 阿里云百炼 API Key（用于 AI 对话）
  • Natapp Authtoken（用于内网穿透）
  • 邮箱账号和授权码（用于发送文件）
