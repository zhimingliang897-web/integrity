Token 消耗对比示例

相同语义问题 "请解释什么是机器学习" 在不同模型下的 Token 消耗：

模型 中文输入 英文输入 图片输入 差异分析
qwen-turbo 156 tokens 203 tokens N/A 中文节省 23%
qwen-plus 148 tokens 195 tokens N/A 中文节省 24%
qwen-max 152 tokens 198 tokens N/A 中文节省 23%
qwen-vl-plus 164 tokens 212 tokens 1,847 tokens 图片消耗最高
gpt-4o 142 tokens 178 tokens 1,523 tokens 中文节省 20%
gpt-4o-mini 138 tokens 172 tokens 1,456 tokens 性价比最高

关键发现

💡 优化建议

  • 💰
    中文输入更省 Token

    相同语义下，中文比英文节省约 20-25% 的 Token 消耗

  • 🖼️
    图片输入成本最高

    图片会转换为大量 Token，成本是文本的 10 倍以上

  • 🎯
    选择合适的模型

    简单任务用 qwen-turbo/gpt-4o-mini，复杂任务用 qwen-max/gpt-4o

  • 📝
    Prompt 优化

    精简 Prompt，避免冗余描述，可显著降低成本
