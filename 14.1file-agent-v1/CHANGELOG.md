# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2024-03-14

### Added
- 移动文件功能：支持将文件移动到其他文件夹
- 回收站操作：添加恢复、永久删除、清空回收站按钮
- Word文档预览：支持 .doc/.docx 文件文本内容预览
- 文件夹树选择器：移动文件时可浏览文件夹结构

### Fixed
- Windows路径转义问题：修复路径中反斜杠被当作转义字符导致预览/下载失败
- 移动文件条件判断逻辑错误
- 创建文件夹失败时无错误提示
- 删除文件失败时无错误提示
- 下载接口缺少路径权限验证（安全修复）
- API参数缺少Query装饰器

### Changed
- 优化文件列表渲染，使用 data-path 属性避免路径转义问题
- 改进错误处理，所有操作都有明确的成功/失败反馈
- 更新 README 文档，添加完整功能说明和API文档

## [1.0.0] - 2024-03-01

### Added
- 文件浏览：文件夹导航、排序、分页
- 文件上传：多文件上传、拖拽上传、大文件支持（最大2GB）
- 文件下载：单文件下载、多文件打包ZIP下载
- 在线预览：图片/视频/音频/PDF/文本文件
- 文件操作：新建文件夹、重命名、复制、删除
- 回收站：移入回收站、查看回收站
- 全文搜索：关键词搜索、按类型分类搜索
- 挂载目录：挂载多个本地目录、只读权限控制
- AI助手：对接阿里百炼LLM，自然语言管理文件
- 内网穿透：通过Natapp实现外网访问
- 用户认证：密码登录、Session管理
