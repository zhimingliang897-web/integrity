# 3d2 学习指南

## 本节内容概览  
本节系统讲解了**Euler公式及其拓扑推广**、**亏格（genus）的定义与几何意义**、**三角网格的组合性质**（特别是平均顶点度数为6）、**显式与隐式3D表示方法**，以及**隐式函数的核心数学性质**（零水平集、梯度即法向、布尔运算等）。这些内容构成了3D几何表示、拓扑分析和微分几何的基础，是理解3D深度学习中形状建模与处理的关键。

---

## 必学考点（按重要性排序）

### ⭐⭐⭐ **1. 扩展Euler公式与亏格的关系**  
- **公式**：对于闭合可定向曲面，$|V| - |E| + |F| = 2 - 2g$，其中 $g$ 为亏格（手柄数）  
- **掌握要求**：能用于计算或证明（如2025年Q2）  
- **时间戳**：[00:08:10–00:12:26]（凸多面体特例 $g=0$），[00:35:37–00:35:55]（一般形式）

### ⭐⭐⭐ **2. 亏格（Genus）的定义与直观理解**  
- **严格定义**：最大非相交简单闭曲线数，沿其切割后仍为单连通  
- **直观理解**：“手柄”数量（球 $g=0$，环面 $g=1$，双环面 $g=2$）  
- **考试形式**：概念题、判断题、简答题  
- **时间戳**：[00:13:04–00:30:03]，[00:30:07–00:30:43]

### ⭐⭐⭐ **3. 三角网格的组合关系与平均顶点度数为6**  
- **核心关系**：  
  - $2|E| = 3|F|$（每条边被两个三角形共享，每个面有3条边）  
  - 顶点度数总和 $= 2|E|$  
  - 结合Euler公式可推导：若所有顶点度数为6（6-正则），则 $g=1$  
- **结论**：闭合流形三角网格的**平均顶点度数趋近于6**  
- **时间戳**：[01:02:47–01:08:48]，[01:10:22–01:15:51]

### ⭐⭐⭐ **4. 隐式函数表示3D几何的核心原理**  
- **零水平集**：$\{ \mathbf{p} \mid f(\mathbf{p}) = 0 \}$ 定义曲面  
- **内外判别**：通常 $f < 0$ 为内部，$f > 0$ 为外部（符号约定）  
- **时间戳**：[02:01:20–02:03:14]，[02:05:56–02:06:02]，[02:30:09–02:31:04]

### ⭐⭐⭐ **5. 隐式曲面的法向量由梯度给出**  
- **公式**：法向 $\mathbf{n} \propto \nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right)$  
- **原理**：沿切向移动时 $f$ 不变 ⇒ 方向导数为0 ⇒ $\nabla f \perp$ 切平面  
- **应用**：SDF/DeepSDF中法向计算、渲染、曲率分析  
- **时间戳**：[02:21:06–02:23:38]，[02:30:09–02:33:10]

### ⭐⭐⭐ **6. 隐式函数的布尔运算**  
- **并（Union）**：$\max(f_1, f_2)$  
- **交（Intersection）**：$\min(f_1, f_2)$  
- **差（Subtraction）**：$\min(f_1, -f_2)$  
- **时间戳**：[02:08:49–02:12:07]

### ⭐⭐ **7. 开曲面的Euler公式（带边界）**  
- **公式**：$|V| - |E| + |F| = 2 - 2g - b$，其中 $b$ 为边界环数  
- **适用对象**：带边界的可定向流形网格（如猫模型）  
- **时间戳**：[00:46:56–00:49:41]

### ⭐⭐ **8. 欧拉示性数的拓扑不变性**  
- **性质**：只要拓扑不变（$g$ 不变），无论网格如何细分，$\chi = |V| - |E| + |F|$ 恒定  
- **意义**：验证网格是否同胚于某标准曲面  
- **时间戳**：[00:41:28–00:43:28]

### ⭐⭐ **9. 三种外观映射技术对比**  
| 技术 | 是否改变几何 | 法线处理 | 轮廓变化 |
|------|---------------|--------|--------|
| 纹理映射 | 否 | 不变 | 无 |
| 凹凸映射 | 否 | 修改法线 | 无（轮廓仍光滑） |
| 位移映射 | 是 | 真实几何改变 | 有（轮廓复杂化） |
- **时间戳**：[01:34:48–01:46:46]，[01:44:59–01:45:33]

### ⭐⭐ **10. 显式 vs 隐式表示**  
- **显式**：点云、网格、参数化曲面 → 几何直接可得  
- **隐式**：标量函数零等值面 → 需算法提取（如Marching Cubes）  
- **时间戳**：[01:58:51–02:00:08]

## 本节关键术语表

| 术语 | 定义 |
|------|------|
| **Euler公式** | 凸多面体满足 $|V| - |E| + |F| = 2$；闭合可定向曲面推广为 $= 2 - 2g$ |
| **亏格（Genus）** | 可定向闭合曲面的“手柄数”，拓扑不变量；等于最大非相交切割曲线数 |
| **欧拉示性数（Euler Characteristic）** | $\chi = |V| - |E| + |F|$，对亏格 $g$ 的闭合曲面，$\chi = 2 - 2g$ |
| **三角网格（Triangle Mesh）** | 所有面为三角形的网格，保证平面性，是图形学标准格式 |
| **顶点度数（Valence）** | 与顶点相连的边数；闭合流形三角网格平均度数≈6 |
| **零水平集（Zero Level Set）** | $\{ \mathbf{p} \mid f(\mathbf{p}) = 0 \}$，隐式函数定义的3D曲面 |
| **隐式函数（Implicit Function）** | $f: \mathbb{R}^3 \to \mathbb{R}$，其零水平集表示几何 |
| **梯度（Gradient ∇f）** | $(\partial f/\partial x, \partial f/\partial y, \partial f/\partial z)$，在隐式曲面上为法向方向 |
| **显式表示** | 几何直接存储（如点云坐标、网格顶点） |
| **隐式表示** | 几何由函数零等值面隐含定义，需提取算法 |
| **布尔运算（隐式）** | Union=max, Intersection=min, Subtraction=min(f₁, -f₂) |
| **开曲面** | 带边界的曲面，Euler公式含边界项 $b$ |
| **闭合曲面** | 无边界曲面（如球、环面） |
| **可定向曲面** | 存在全局一致法向（如球、环面），排除莫比乌斯带等 |
