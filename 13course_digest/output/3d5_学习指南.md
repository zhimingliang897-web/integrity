# 3d5 学习指南

## 本节内容概览  
本学习指南整合了课程第五周及复习课中关于**3D表示方法**（传统与神经）、**微分几何基础**（曲线与曲面）以及**网格拓扑性质**的核心内容，聚焦于2025年Quiz的考试范围。重点包括四种传统3D表示、DeepSDF/IF-Net/NUDF等神经隐式方法、弧长参数化、曲率与挠率、高斯曲率符号判断、6-正则三角网格的拓扑推导等。明确排除非可定向曲面和神经参数曲面等内容。

---

## 必学考点（按重要性排序）

### ⭐⭐⭐ **1. 四种传统3D表示与神经隐式函数（DeepSDF, IF-Net, NUDF）**
- **[00:03:32–00:04:23]** 需掌握：点云（无序XYZ点集）、网格（顶点+面片+连接性）、参数函数、隐式函数（如SDF）；神经隐式函数包括DeepSDF、IF-Net、NUDF；**神经参数曲面不考**。
- **[01:46:00–01:50:21]** 隐式曲面定义为标量函数的零水平集（SDF/UDF用 $f(p)=0$，Occupancy用 $f(p)=0.5$）。
- **关联真题**：2025 Q1。

### ⭐⭐⭐ **2. DeepSDF原理及其扩展至开放曲面（NUDF）**
- **[00:09:06–00:10:12]** DeepSDF使用**潜在编码**（latent code）作为网络输入，通过**auto-decoder架构**表示多类物体；训练和推理时均需优化该编码。
- **[00:12:28–00:13:10]** SDF仅适用于**封闭曲面**（有内外之分），而UDF（无符号距离场）可表示**开放曲面**（值恒≥0）。
- **[02:10:35–02:14:35]** 扩展DeepSDF以支持开放曲面 → 替换SDF为UDF。
- **[00:30:00–00:31:46]** UDF在零等值面**不可导**，导致训练困难（需在方案设计中考虑）。
- **关联真题**：2025 Q5。

### ⭐⭐⭐ **3. 微分几何：曲线的曲率与挠率**
- **[00:34:51–00:35:39]** **弧长参数化**：速度向量模为1（$\|x'(s)\|=1$），简化曲率计算。
- **[00:37:05–00:38:04]** **曲率 $\kappa = \|x''(s)\|$**，衡量弯曲程度；直线曲率为0。
- **[00:38:07–00:41:05]** **挠率 $\tau$** 衡量空间曲线偏离平面的程度；**平面曲线 $\tau=0$**，螺旋线为典型空间曲线（$\tau \neq 0$）。
- **[02:04:05–02:05:14]** 对参数曲线 $\alpha(t)$，需能计算一阶导（切向量）和二阶导（加速度）在指定点（如 $t=2\pi$）的值。
- **关联真题**：2025 Q3。

### ⭐⭐⭐ **4. 高斯曲率的定义与几何意义**
- **[01:11:58–01:13:52]** 高斯曲率 $K = \kappa_1 \cdot \kappa_2$：
  - $K > 0$：局部球状（椭圆点，如环面外侧）；
  - $K = 0$：柱状或平面（抛物点）；
  - $K < 0$：鞍状（双曲点，如环面内侧）。
- **[02:07:15–02:10:26]** 能根据局部形状判断 $K$ 的符号。
- **关联真题**：2025 Q4。

### ⭐⭐⭐ **5. 6-正则三角网格的拓扑性质（必为环面）**
- **[01:50:27–02:00:06]** 对封闭流形三角网格：
  - 握手定理：总度数 $= 2|E|$；
  - 若每个顶点度数为6 → $6|V| = 2|E| \Rightarrow |E| = 3|V|$；
  - 欧拉公式：$|V| - |E| + |F| = 2 - 2g$；
  - 结合 $2|E| = 3|F|$ 可得 $g = 1$（即亏格为1，拓扑同胚于环面）。
- **[02:00:06–02:02:30]** 此推导是标准证明题模板。
- **关联真题**：2025 Q2。

### ⭐⭐ **6. IF-Net的改进：多尺度特征提升鲁棒性**
- **[00:11:35–00:12:24]** IF-Net用**多尺度特征**（而非原始坐标）作为网络输入，使表示对旋转/缩放不变，克服DeepSDF的敏感性问题。
- **可能考法**：与DeepSDF比较优劣。

### ⭐⭐ **7. 正则曲面条件与直线的几何性质**
- **[00:50:55–00:53:19]** 正则曲面要求偏导向量 $S_U$ 与 $S_V$ **线性无关**（否则法向量未定义）。
- **[00:32:18–00:33:58]** 直线是**平面曲线**（存在包含它的平面）。

## 本节关键术语表

| 术语 | 定义 |
|------|------|
| **潜在编码 (Latent Code)** | 高维向量，作为DeepSDF网络输入以区分不同3D物体实例 |
| **解码器架构 (Auto-Decoder)** | DeepSDF训练方式：直接优化每个物体的潜在编码，无需独立编码器 |
| **有符号距离场 (SDF)** | 隐式函数，符号表示内外，零等值面为封闭曲面 |
| **无符号距离场 (UDF)** | 距离恒为非负，可表示开放曲面，但在零等值面不可导 |
| **多尺度特征 (Multi-scale Features)** | IF-Net中提取的层次化几何特征，提升对旋转/缩放的鲁棒性 |
| **弧长参数化** | 以曲线长度为参数，满足 $\|x'(s)\| = 1$ |
| **曲率 (Curvature)** | $\kappa = \|x''(s)\|$，衡量局部弯曲程度 |
| **挠率 (Torsion)** | $\tau$，衡量空间曲线偏离平面的程度；平面曲线 $\tau = 0$ |
| **高斯曲率 (Gaussian Curvature)** | $K = \kappa_1 \cdot \kappa_2$，符号决定局部几何类型 |
| **主曲率 (Principal Curvatures)** | 法曲率的极值 $\kappa_1, \kappa_2$，对应正交主方向（若不等） |
| **Meusnier 定理** | 同一切方向上所有曲面曲线具有相同法曲率 |
| **6-正则网格** | 每个顶点度数为6的三角网格 |
| **欧拉示性数** | $\chi = |V| - |E| + |F| = 2 - 2g$，$g$ 为亏格 |
| **正则曲面** | 偏导向量 $S_U, S_V$ 线性无关，确保法向量良好定义 |
| **隐式曲面** | 由 $f(p) = 0$ 定义的曲面（$f: \mathbb{R}^3 \to \mathbb{R}$） |
