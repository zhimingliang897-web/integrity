## 二、分章节复习路径

### 模块一：基础概念 【必学】

**核心概念：**
- **Weak Stationarity（弱平稳性）** [3.mp4 00:20:42-00:23:00]
  - 均值恒定：μ_t = μ（不依赖 t）
  - 自协方差只依赖 lag k：γ_k = Cov(X_t, X_{t+k})
  
- **ACVF（自协方差函数）** [3.mp4 00:01:21-00:02:05]
  - γ_k = Cov(X_t, X_{t+k}) = E[(X_t - μ)(X_{t+k} - μ)]
  
- **ACF（自相关函数）**
  - ρ_k = γ_k / γ_0

**关键公式：**
```
Sample ACVF: γ̂_k = (1/n) Σ(X_t - X̄)(X_{t+k} - X̄)
Sample ACF:  ρ̂_k = γ̂_k / γ̂_0
```

**重要性质：** [3.mp4 00:07:53-00:11:00]
- γ_{-k} = γ_k（对称性）
- γ_0 = Var(X_t)
- ρ_0 = 1

**典型应用：**
- 使用 Correlogram 判断短期/长期记忆
- 判断序列是否平稳

### 模块二：ARMA 模型 【必学】

#### 2.1 AR(p) 模型

**定义：**
```
X_t = φ₁X_{t-1} + φ₂X_{t-2} + ... + φ_pX_{t-p} + Z_t
```

**ACF 特征：** 指数衰减（或阻尼正弦波）

**PACF 特征：** 在 lag p 后截尾（φ_{kk} = 0 for k > p）

#### 2.2 MA(q) 模型

**定义：**
```
X_t = Z_t + θ₁Z_{t-1} + θ₂Z_{t-2} + ... + θ_qZ_{t-q}
```

**ACF 特征：** 在 lag q 后截尾（ρ_k = 0 for k > q）

**PACF 特征：** 指数衰减

#### 2.3 ARMA(p,q) 模型

**定义：**
```
X_t - φ₁X_{t-1} - ... - φ_pX_{t-p} = Z_t + θ₁Z_{t-1} + ... + θ_qZ_{t-q}
```

**ACF/PACF 特征：** 两者都呈指数衰减

**关键概念：**
- **Causality（因果性）**：X_t 只依赖于过去和现在的噪声 [5.mp4 00:09:50-00:11:55]
- **Invertibility（可逆性）**：MA 部分可表示为 AR(∞)

**ACVF 计算方法：** [5.mp4 00:12:11-00:16:00]

| 方法 | 适用场景 | 步骤 |
|------|----------|------|
| **Step-by-step** | 低阶模型 | 对 k=0,1,...,q，乘以 X_{t-k} 取期望，解方程组 |
| **Causal representation** | 系统性计算 | 将模型写成 MA(∞) 形式，用 γ_k = σ²Σψ_jψ_{j+k} |

### 模块三：参数估计与模型识别 【必学】

#### 3.1 PACF（偏自相关函数）【核心考点】

**定义：** [6.mp4 00:16:06-00:20:08]
- φ_{11} = ρ_1
- 对于 k ≥ 2：
  ```
  φ_{kk} = Corr(X_{t+k} - X̂_{t+k}, X_t - X̂_t)
  ```
  其中 X̂ 是基于 X_{t+1},...,X_{t+k-1} 的最佳线性预测

**回归表示：** [6.mp4 00:27:12-00:28:57]
```
对 X_t 关于 X_{t-1}, X_{t-2}, ..., X_{t-k} 做回归
φ_{kk} = 最后一个回归系数 α̂_{kk}
```

**物理意义：** φ_{kk} 衡量 X_{t-k} 对 X_t 的**边际贡献**（去除中间滞后项影响后）

#### 3.2 Yule-Walker 方程

**用于 AR(p) 参数估计：**
```
γ_k = φ₁γ_{k-1} + φ₂γ_{k-2} + ... + φ_pγ_{k-p},  k ≥ 1
```

#### 3.3 模型选择准则

**AIC（Akaike Information Criterion）：**
```
AIC = -2 log L + 2k
```

**BIC（Bayesian Information Criterion）：**
```
BIC = -2 log L + k log n
```

**选择原则：** AIC/BIC 值越小，模型越好

#### 3.4 Box-Jenkins 方法 【核心框架】[6.mp4 00:11:40-00:13:14]

```
┌─────────────────────────────────────────────────────┐
│  Step 1: Model Identification (ACF/PACF 分析)       │
│     ↓                                               │
│  Step 2: Order Selection (AIC/BIC, 统计检验)        │
│     ↓                                               │
│  Step 3: Parameter Estimation (MLE, Yule-Walker)    │
│     ↓                                               │
│  Step 4: Residual Analysis (诊断检验)               │
│     ↓                                               │
│  Step 5: Forecasting                                │
└─────────────────────────────────────────────────────┘
```
