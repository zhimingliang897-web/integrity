## 二、分章节复习路径

### 模块一：数学基础

#### 📌 Lecture 1: Linear Algebra (Part 1 & 2) 【必学】

**核心概念：**
- 矩阵运算（乘法、转置、逆矩阵）
- Orthonormal Matrix（正交归一矩阵）
- 矩阵的几何意义：变换（拉伸、旋转、翻转）

**关键公式：**
```
正交矩阵性质：A·Aᵀ = Aᵀ·A = I，即 A⁻¹ = Aᵀ
```

**典型应用：**
- 理解神经网络中的权重矩阵变换
- PCA降维的数学基础

---

#### 📌 Lecture 2: Introduction to Probability 【重要】

**核心概念：**
- 条件概率、贝叶斯定理
- 概率分布（离散/连续）
- 期望、方差

**典型应用：**
- 理解机器学习中的不确定性
- 逻辑回归的概率解释

---

#### 📌 Lecture 3: Information Theory 【重要】

**核心概念：**
- Entropy（熵）、Cross-Entropy（交叉熵）
- KL Divergence（KL散度）

**关键公式：**
```
H(X) = -Σ p(x) log p(x)
Cross-Entropy Loss = -Σ y log ŷ
```

**典型应用：**
- 分类任务的损失函数
- 模型评估指标

---

### 模块二：机器学习基础

#### 📌 Lecture 3: Linear Regression & Ridge Regression 【必学】

**核心概念：**
- 线性回归的闭式解
- 过拟合与正则化
- Ridge Regression 的 L2 正则化

**关键公式：**
```
线性回归解：β = (XᵀX)⁻¹Xᵀy
Ridge回归：β = (XᵀX + λI)⁻¹Xᵀy
```

---

#### 📌 Lecture 3: Logistic Regression 【必学】

**核心概念：**
- Sigmoid 函数
- 二分类问题
- 最大似然估计

**关键公式：**
```
ŷ = σ(βᵀx)，其中 σ(z) = 1/(1+e⁻ᶻ)
Loss = -[y log ŷ + (1-y) log(1-ŷ)]
```

---

### 模块三：神经网络基础

#### 📌 Week 5: Multi-layer Perceptron (MLP) 【必学】

**核心概念：**
- 前向传播
- 反向传播
- 激活函数

**关键公式：**
```
z = W·x + b
a = σ(z)
```

**典型应用：**
- 多层特征学习
- Representation Learning 的核心思想

---

#### 📌 Week 5: Multi-class Classification & Cross-entropy Loss 【必学】

**核心概念：**
- Softmax 函数
- 多分类交叉熵损失

**关键公式：**
```
Softmax: p(y=i) = eᶻⁱ / Σⱼ eᶻʲ
Loss = -Σ yᵢ log ŷᵢ
```

---

### 模块四：卷积神经网络

#### 📌 Week 5: Convolutional Networks 【必学】

**核心概念：**
- 卷积操作
- Pooling（Max/Average）
- ReLU 激活函数

**关键参数：**
- Kernel size, Stride, Padding
- Feature map 尺寸计算

---

#### 📌 Week 5: ResNet Architecture 【必学】

**核心概念：**
- Skip Connection（跳跃连接）
- 解决梯度消失问题
- Residual Block

**关键公式：**
```
y = F(x) + x  (Residual learning)
```

---

#### 📌 Week 5: Batch Normalization 【必学】

**核心概念：**
- Internal Covariate Shift
- BN 层的位置和作用
- 训练/测试时的行为差异

**关键公式：**
```
BN(x) = γ · (x - μ) / σ + β
```

---

#### 📌 Week 7: Advanced CNNs 【重要】

**核心概念：**
- Width/Depth/Resolution 的权衡
- Depthwise Separable Convolution
- Grouped Convolution
- DenseNet、FractalNet
- 3D Convolution for Video

**典型架构：**
- Wide ResNet
- EfficientNet
- C3D, I3D, S3D

---

### 模块五：优化与正则化

#### 📌 Week 6 & 9: Optimization 【必学】

**核心概念：**
- Gradient Descent 变体
- Learning Rate 调度策略
- Momentum

---

#### 📌 Week 9: Regularization (Part 2) 【必学】

**核心概念：**
- Overfitting vs Underfitting
- L1/L2 Regularization
- Dropout
- Data Augmentation
- Early Stopping

**关键理解：**
```
Deep Learning = Representation Learning
深层网络自动学习特征表示
```
