## 四、做题与回顾建议

### 样题分析（基于提供的 7 道样题）

| 题号 | 知识点 | 解题要点 |
|------|--------|----------|
| Q1 | 多项分布概率 | 使用多项分布公式 $P = \frac{n!}{x_1! \cdots x_k!} p_1^{x_1} \cdots p_k^{x_k}$ |
| Q2 | 正定矩阵 | 正定 → 所有特征值 > 0；半正定 → 所有特征值 ≥ 0 |
| Q3 | 概率公理 | 概率不能为负；样本空间概率和为 1；伯努利分布描述单次试验 |
| Q4 | MLE | MLE 方差估计是有偏的（分母为 n），非唯一正确估计 |
| Q5 | 逻辑回归 | 是线性模型（决策边界线性），无法表示 XOR |
| Q6 | 高斯 PDF | 注意公式中分母是 $2\sigma^2$，不是 $\sigma$ |
| Q7 | 熵计算 | $H = -\sum p_i \log_2 p_i$，结果为正数 |

### 常见陷阱 ⚠️

| 陷阱 | 说明 |
|------|------|
| **高斯 PDF 公式** | 分母是 $\sqrt{2\pi}\sigma$，指数部分分母是 $2\sigma^2$（样题 Q6 的错误选项漏掉了 2） |
| **MLE 方差估计** | MLE 方差估计 $\hat{\sigma}^2 = \frac{1}{n}\sum(x_i-\bar{x})^2$ 是有偏的，非"唯一正确" |
| **伯努利 vs 二项** | 伯努利描述单次试验，二项分布描述多次独立试验 |
| **正定 vs 半正定** | 正定要求特征值严格大于 0，半正定允许等于 0 |
| **熵的符号** | 熵始终为正数，公式中有负号但结果为正 |
| **逻辑回归线性性** | 逻辑回归是线性模型（决策边界线性），非线性的是 sigmoid 激活 |

### 自行构造练习题建议

由于样题有限，建议按以下方式自行构造：

1. **概率计算题**：改变多项分布的参数，重新计算概率
2. **熵计算题**：构造不同的离散分布，计算熵
3. **概念判断题**：针对每个考点，写出 2-3 条正确陈述和 1-2 条错误陈述，自我辨析

## 五、重点公式速查表

| 公式名称 | 公式 |
|----------|------|
| 高斯 PDF | $f(x|\mu,\sigma) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$ |
| 熵 | $H(p) = -\sum_i p_i \log_2 p_i$ |
| 正定矩阵 | $x^T A x > 0, \forall x \neq 0$ |
| 半正定矩阵 | $x^T A x \geq 0, \forall x$ |
| MLE 均值估计 | $\hat{\mu} = \frac{1}{n}\sum_{i=1}^n x_i$ |
| MLE 方差估计 | $\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \hat{\mu})^2$（有偏） |
