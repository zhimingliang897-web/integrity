# 3d1 学习指南

## 本节内容概览  
本课程导论系统介绍了**3D深度学习的发展脉络（2014–2025）**、**四大核心问题（表示、重建、生成、分析）**，以及**主流3D表示方法（点云、网格、体素、多视图、隐式函数）的原理、优劣与适用场景**。重点突出了**PointNet（2017）** 和 **DeepSDF（2019）** 的里程碑意义，并明确了**开卷测验仅覆盖第一模块：3D表示**，包括传统表示、神经隐式表示及微分几何基础。课程强调“**最佳表示取决于应用**”，并深入讲解了三角网格的逼近误差、拓扑表达能力、流形性、可定向性等关键性质。

## 必学考点（按重要性排序）

### ⭐⭐⭐ 核心必考（高频简答/证明题）
- **[00:16:33–00:20:29] DeepSDF的核心思想**：单个神经网络通过**隐向量（latent code）** 编码单个物体身份，实现对整个形状类别的统一表示。考试可能要求解释其如何解决多物体建模问题（如2025 Q5(a)）。
- **[01:39:52–01:49:22] 三角网格的二次收敛误差**：当平均边长为 $ h $ 时，对光滑曲面的逼近误差为 $ O(h^2) $。需掌握推导思路（圆弧-弦长+泰勒展开），并能解释“边长减半 → 误差降至1/4”（如2025 Q1(b)潜在考点）。
- **[02:03:16–02:07:38] 流形网格定义**：(1) 每条边最多邻接两个面；(2) 每个顶点邻域同胚于圆盘（内部）或半圆盘（边界）。考试可能要求判断网格是否流形或解释原因（如2025 Q2前提）。
- **[02:09:18–02:15:04] 可定向网格定义**：所有面的顶点排序（CW/CCW）可全局一致，且每条内部边在相邻两面中方向相反。考试可能要求解释或判断可定向性（如2025 Q4背景知识）。
- **[00:08:07–00:13:53] 体素化的局限性**：空间复杂度 $ O(n^3) $，导致高分辨率内存爆炸。需能与其他表示对比（如2025 Q1(a)(b)）。

### ⭐⭐ 重要考点（比较题/概念辨析）
- **[01:18:57–01:22:02] 3D表示的选择依据**：点云适合采集、网格适合渲染、隐式函数适合重建。考试可能以简答题形式问“为何某任务选某表示”（如2025 Q1(b)）。
- **[01:28:56–01:30:08] 网格 vs 点云**：网格通过显式连接性消除点云的几何歧义，是实时渲染标准输入（对应2025 Q1(b)）。
- **[01:55:37–01:56:29] 网格的任意拓扑表达能力**：可表示任意 genus（洞数）、带/无边界曲面。需结合 Euler 公式解题（如2025 Q2）。
- **[00:40:02–00:40:13] SDF vs UDF**：SDF 有符号，仅支持闭合曲面；UDF 无符号，支持开放/非流形曲面（如2025 Q5(b)）。
- **[01:22:48–01:24:35] 点云的性质**：无序集合（排列不变性）、缺乏连接性 → 导致几何歧义。
- **[02:01:25–02:02:35] 边界边与开/闭网格**：边界边仅属一个面；闭网格无边界边（2025 Q2关键前提）。
- **[00:16:18–00:18:43] PointNet的突破**：首个直接处理无序点云的网络，无需体素化或多视图，奠定点云学习基础。

### ⭐ 基础考点（背景知识/定义）
- **[00:27:05–00:29:13] 3D深度学习四大问题**：表示、重建、生成、分析 —— 需掌握基本定义。
- **[00:45:22–00:45:42] 测验范围**：仅限第一模块（3D表示），含传统表示、神经表示（如DeepSDF）、微分几何基础。
- **[02:30:13–02:33:16] 莫比乌斯带单侧性**：沿中心线剪开得一个更长单侧环（非两个环）。
- **[02:33:29–02:35:19] 莫比乌斯带剪裁特性**：沿距边界1/3处剪开得两个相连环（一长一短）。
- **[02:35:37–02:37:35] 可定向 vs 非可定向曲面**：可定向曲面双侧（如球、环面），非可定向单侧（如莫比乌斯带）。

## 本节关键术语表

| 术语 | 定义 |
|------|------|
| **体素化 (Voxelization)** | 将3D空间划分为规则立方体网格，内存消耗随分辨率立方增长（$O(n^3)$） |
| **多视图投影 (Multi-view Projection)** | 从多视角渲染3D为2D图像供2D CNN处理，但丢失深度与背面几何信息 |
| **隐向量 (Latent Code)** | DeepSDF中编码单个物体身份的低维向量，使单网络表示整类形状 |
| **PointNet** | 首个直接处理无序点云的网络，通过置换不变性解决点序问题 |
| **3D表示 (3D Representation)** | 描述3D物体几何的数字形式，如点云、网格、SDF等 |
| **有符号距离场 (SDF)** | 隐式函数，值为点到曲面最短距离，符号区分内外，仅支持闭合曲面 |
| **无符号距离场 (UDF)** | 隐式函数，值恒为非负最短距离，支持开放/非流形曲面 |
| **微分几何基础** | 分析3D形状局部性质的工具：曲线曲率/挠率、曲面高斯曲率等 |
| **点云 (Point Cloud)** | 无序3D点集，缺乏连接信息，具排列不变性 |
| **排列不变性** | 点云性质：点顺序不影响整体几何表示 |
| **有向点云** | 每个点附带法向量，提升重建与渲染质量 |
| **网格 (Mesh)** | 由顶点、边、面构成，通过显式连接提供连续表面近似 |
| **隐式函数** | 通过 $ f(x,y,z)=0 $ 定义曲面，符号区分区域 |
| **分段线性逼近** | 用三角形拼接近似光滑曲面，误差 $ O(h^2) $ |
| **二次收敛率** | 逼近误差与平均边长 $ h $ 平方成正比 |
| **任意拓扑** | 网格可表示任意 genus（洞数）和边界数量 |
| **自适应采样** | 在几何复杂处加密顶点，平滑处稀疏，平衡精度与效率 |
| **GPU友好** | 三角网格是图形硬件渲染管线原生输入格式 |
| **顶点度 (Valence)** | 与一顶点相连的边数 |
| **边界边** | 仅属于一个多边形面的边 |
| **流形网格** | 满足：(1) 每边≤2面；(2) 顶点邻域为圆盘/半圆盘 |
| **可定向网格** | 所有面顶点排序可全局一致，内部边方向相反 |
| **莫比乌斯带** | 非可定向、单边界、单侧曲面，扭转粘合形成 |
| **可定向曲面** | 存在一致法向量场（如球面、环面） |
| **单侧曲面** | 存在路径使“正面”连续变“反面”（如莫比乌斯带） |
